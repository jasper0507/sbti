# STTI（学生孪梗指标）

面向大学生的校园梗娱乐问卷静态站：纯 HTML / CSS / JS，在浏览器本地计分与展示结果。

## 本地预览

用任意静态服务器打开项目根目录（或直接双击 `index.html`，部分浏览器对剪贴板 API 可能有限制）。

## GitHub Pages

推送至 `main` 后，工作流会自动部署：

1. 在仓库 **Settings → Pages** 中，将 **Build and deployment → Source** 设为 **GitHub Actions**（首次需手动选择一次）。
2. 发布后访问：**`https://<你的用户名>.github.io/STTI/`**（与仓库同名路径）。

若你希望使用自定义域名，在同一 Pages 设置里绑定域名即可。
